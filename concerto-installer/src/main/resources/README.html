XWoot Overview

XWoot is the prototype of XWiki Concerto. This is an early release, implementing base P2P services for storing patches in a distributed network, and a data reconciliation sevice based on the WOOT algorithm.

This implementation is based on the architecture and specifications from the L2-2 deliverables from August 2007, section 2.4.1.2. Thus, it is based on:

  • The WOOT (WithOut Operational Transformation) algorithm for the synchronization of textual content;
  • The LPBCast algorithm for propagating messages between different peers;
  • The JLibDiff library for computing the differences between two texts;
  • A Thomas Rule engine for synchronizing XWiki metadata.
